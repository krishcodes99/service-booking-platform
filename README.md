# Service Booking Platform

A web-based service booking platform developed using PHP Laravel and MySQL. This application allows users to browse services, book appointments, and manage bookings efficiently.

## 🚀 Features
- User registration and login
- Service listing and details
- Booking system with date/time selection
- Admin panel for managing services and bookings
- Database integration with MySQL

## 🛠️ Tech Stack
- PHP (Laravel Framework)
- MySQL
- HTML, CSS, JavaScript

## 📂 Installation
1. Clone the repository
2. Run `composer install`
3. Set up `.env` file
4. Run `php artisan migrate`
5. Start server using `php artisan serve`
